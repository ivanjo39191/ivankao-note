# Upgrading Node.js to latest version

Created: February 1, 2022 5:22 PM
Tags: Vue

### Ubuntu Linux/Mac:

The module `n` makes version-management easy:

```
sudo npm install n -g

```

For the latest stable version:

```
sudo n stable

```

For the latest version:

```
sudo n latest
```
